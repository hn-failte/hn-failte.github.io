CSS选择符的权重

# CSS选择符的权重

# 目录

  • css选择符权重
    • css选择器权重列表
    • CSS选择符冲突处理

# css选择符权重

# css选择器权重列表 

选择器		权重
内联样式		1000
ID			0100
CLASS		0010
属性选择符	0010
TAG			0001
伪元素		0001
伪类			0001
inhref		0000
群组选择符	不变
后代选择符	相加

# CSS选择符冲突处理

1、权重不同：选择符的css发生冲突时，拥有高权值的css选择符将会显示

2、权重相同：选择符权重一样，将会按照加载顺序进行覆盖（就近原则）

3、a和a:link同时存在时，以a:link优先，注：使用a的伪类时，需要四个伪类(a:link、a:hover、a:active、a:visited)同时使用

4、!important的权重为正无穷
